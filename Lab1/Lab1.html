Bài 1. Viết hàm cost function¶

Nhắc lại, từ công thức tính mean squared error, ta có: $$\mathcal{E}(\hat{y}, y) = \frac{1}{2N} \sum_{i=1}^N (\hat{y}^{(i)}-y^{(i)})^2 $$

$$\hat{y} = w^{T} x $$

Khai triển, kết quả có là $$\mathcal{E}(\hat{y}, y) = \frac{1}{2N} \sum_{i=1}^N (w_1 x_1^{(i)} + w_2 x_2^{(i)} + b - y^{(i)})^2 $$

def np_cost(w1, w2, b, X, y):  #w1 and w2 are parameters
    '''
    Evaluate the cost function in a non-vectorized manner for 
    inputs `X` and targets `y`, at weights `w1`, `w2` and `b`.
    '''
    
    costs = 0
    N = len(y)
    for i in range(len(y)):
        
        # TODO: complete below expression to calculate cost function
        y_hat = w1 * X[i,0] + w2 * X[i,1] + b
        costs += (y_hat - y[i])**2
        
    return costs / (2 * N)
